# AI创作社区平台 — 开发计划

## 项目概述
为开发者打造一个类小红书+QQ空间+微信+小米笔记混合的AI创作社区平台，支持AI对话、代码编辑、论坛浏览、私信社交、个人主页等功能。

## 用户参考链接分析
- 暗色主题（dark theme）风格，类似Claude UI
- 卡片式设计、现代简洁
- 需要低饱和度、温暖的暗色调

## 第一性原理需求分析

### 核心本质
用户需要的不是一堆功能的堆砌，而是一个**围绕AI创作的社交生态系统**：
1. **创作入口**（便签/笔记）— 快速记录想法，让AI协助完成
2. **AI协作**（对话）— 与AI共同创作，嵌入代码/富文本
3. **社区发现**（论坛）— 双列流发现他人作品，一键复刻
4. **社交连接**（私信）— 创作者之间的交流与转发
5. **自我表达**（个人页）— 展示个人创作与状态

### 补充需求（用户未考虑到）
1. **统一导航系统** — 底部Tab栏 + 顶部状态栏
2. **搜索发现** — 全局搜索（用户/帖子/话题/代码）
3. **通知中心** — 红点提醒、系统通知、互动提醒
4. **创作工具栏** — 悬浮快捷操作（+号FAB）
5. **主题切换** — 支持暗色/浅色（基于链接参考，默认暗色）
6. **离线状态/网络检测** — 服务器状态的可视化需要网络感知
7. **空状态设计** — 每个页面的无数据状态
8. **骨架屏加载** — 论坛流和便签流的加载体验
9. **下拉刷新/上拉加载** — 标准的移动端交互
10. **动画过渡** — 页面切换、卡片展开、点赞动效
11. **手势操作** — 左滑删除/收藏、双击点赞、长按菜单
12. **输入框富文本工具栏** — 加粗、代码块、链接、图片
13. **分享面板** — 系统级分享、链接复制、生成海报
14. **设置页** — 账号、隐私、通知、关于、退出

## Stage 2 — Web原型开发（vibecoding-webapp-swarm）
使用React + Tailwind CSS + shadcn/ui 开发高保真交互原型

### 页面清单
1. **首页/便签流** — 类小米笔记的无限滚动便签卡片 + 顶部服务器状态面板
2. **AI对话页** — 折叠布局，顶部AI头像抽屉式展开，支持代码块、富文本、引用
3. **论坛流** — 小红书式双列瀑布流，帖子卡片，红心收藏，一键同款
4. **帖子详情** — Reddit式层级评论，锐评跟评，评论折叠/展开
5. **私信/好友** — 抖音式好友列表，对话页支持转发、富媒体
6. **个人主页** — QQ秀虚拟形象 + QQ空间动态流 + 我的作品网格
7. **设置页** — 账号管理、主题、通知、隐私
8. **搜索页** — 全局搜索，多标签结果
9. **通知页** — 各类通知聚合

### 技术栈
- React 18 + TypeScript
- Tailwind CSS
- shadcn/ui 组件
- Framer Motion 动画
- Zustand 状态管理
- React Router 页面路由
- Lucide React 图标
